Exercise 6:

CSS Positioning

__

1. Absolute Positioning: Create a webpage with 12 different sized boxes positioned at

various places in the document using position: absolute and CSS classes or IDs.

2. Fixed Positioning: Create a webpage with 6 different sized boxes and 8 paragraphs of

Lorem Ipsum text. The text must scroll vertically while the boxes must be remain fixed

in place. Additionally, at least 1 box must appear behind the text and 1 must appear in

front of the text.

3. Floats and Relative Positioning: Position and float can be used together. Set up a

recurring module of red bars with float. Use position: relative and top rules to
